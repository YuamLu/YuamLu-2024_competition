# 三四、其他事項

自109年1月起爆發新型冠狀病毒肺炎(COVID-19)，由於疫情蔓延，各國政府陸續實施各項防疫措施，致使合併公司業務略受影響，全球經濟情勢仍不確定，合併公司採取相關因應措施，除與客戶及廠商繼續保持密切聯繫外，更致力於加強員工健康管理，以減緩對公司營運之衝擊，惟公司實際可能受影響程度，仍需視疫情後續發展而定。
