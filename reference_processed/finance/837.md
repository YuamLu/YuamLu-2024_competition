# 一、公司沿革與業務範圍

瑞昱半導體股份有限公司（以下簡稱「本公司」）設立於民國76年10月21日，並於民國77年3月開始營業，民國78年10月28日遷入新竹科學園區，本公司及子公司（以下統稱「本集團」），主要營業項目為研究、開發、生產、製造、銷售各種積體電路及其相關應用軟體等。
